About The Project

Low-cost telemedicine technology for reliable remote patient care.

The ESP32-Based Telemedicine Monitoring System is designed to detect health issues early and support timely intervention. It continuously measures a patient’s vital parameters, stores them in the cloud, and presents them in an accessible web interface for real-time healthcare monitoring.

What it monitors

The system tracks heart rate, blood oxygen saturation, body temperature, and patient movement, while also identifying sudden falls for emergency response support.

Why it matters

It improves visibility for caregivers and medical staff, especially when patients are at home, recovering after hospitalization, or living in remote and low-resource areas.

Where it helps most

  • Elderly care and fall monitoring
  • Post-hospitalization observation
  • Remote healthcare services
  • Rural and low-resource communities

Key Features

Core capabilities that make the system practical and responsive.

HR

Heart Rate Monitoring

Continuous pulse tracking helps detect cardiovascular changes and supports daily remote observation.

O2

SpO2 Monitoring

Blood oxygen levels are measured in real time to highlight respiratory issues or oxygen drops quickly.

TMP

Contactless Temperature

The MLX90614 sensor enables non-contact temperature measurement for safe and convenient fever tracking.

MOV

Movement Analysis

Patient movement is monitored to understand activity levels, inactivity, and changes in posture.

ALR

Automatic Fall Detection

Acceleration and orientation data are used to detect sudden falls and trigger emergency alerts.

RTC

Accurate Timestamp Logging

The DS3231 RTC module records exact time information for every measurement and event.

System Overview

Each hardware module has a clear role in the monitoring pipeline.

MAX30102 Sensor

Measures heart rate and SpO2 continuously, giving instant insight into oxygen saturation and pulse rate.

MLX90614 Sensor

Captures body temperature without direct contact, supporting hygienic and comfortable measurements.

MPU6050 Sensor

Tracks acceleration and orientation to analyze movement and identify possible fall incidents.

DS3231 RTC Module

Adds precise timestamps to each reading so healthcare data can be logged and reviewed accurately.

How It Works

From sensor collection to real-time dashboard visibility.

01

Data Collection

Sensors continuously gather patient vitals, body temperature, and motion information.

02

ESP32 Processing

The microcontroller organizes readings, checks thresholds, and evaluates fall conditions.

03

Timestamp Recording

The RTC module stores the exact time of each event for traceable health monitoring.

04

Cloud Transmission

Processed data is sent wirelessly to Firebase using Wi-Fi connectivity from the ESP32.

05

Dashboard Access

The webpage reads live values and gives doctors, caregivers, and families a clear status view.

06

Alert Response

Abnormal values or falls can trigger alerts for fast attention and better patient safety.
